# Dealzo - Second-Hand Vehicle Marketplace

A modern, feature-rich web application for buying and selling second-hand vehicles with user authentication, real-time chat, wishlists, and comprehensive admin management.

## Features

### 👤 User Features
- **User Authentication** - Register, login, and manage user accounts
- **Browse Vehicles** - Search and filter vehicles by brand, price, year, fuel type, and location
- **Vehicle Details** - View detailed information, images, and specifications
- **Chat System** - Real-time messaging with vehicle sellers
- **Wishlist** - Save favorite vehicles for later
- **Notifications** - Track messages and updates
- **Guest Access** - Browse vehicles without creating an account

### 🛠️ Admin Features
- **Vehicle Management** - Add, edit, and delete vehicle listings
- **Chat Management** - Communicate with users and manage inquiries
- **Admin Dashboard** - Central hub for all administrative tasks
- **User Monitoring** - Track user activities and interactions

## Technologies Used

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Data Storage**: LocalStorage (Browser-based)
- **Styling**: Responsive CSS with modern design patterns
- **Architecture**: Static site suitable for Vercel deployment

## How to Use

### As a User
1. Visit the homepage
2. Browse vehicles or login to use full features
3. Search and filter vehicles by various criteria
4. View vehicle details and chat with sellers
5. Add vehicles to your wishlist
6. Check notifications for messages

### As an Admin
1. Use credentials: `username: admin` | `password: admin123`
2. Manage vehicle listings (add, edit, delete)
3. View and respond to user messages
4. Monitor all platform activities

## Getting Started

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Use demo credentials or create a new account

### Deployment (Vercel)
1. Push code to GitHub
2. Connect repository to Vercel
3. Deploy with one click
4. Your app is live! 🎉
